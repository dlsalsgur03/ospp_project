## 1. 프로젝트 개요
교내 BLE 센서 데이터를 모바일로 수집하고 API로 서버에 전송하는 시스템을 완성했습니다.
데이터 위에서 무엇을 할지 다양한 아이디어가 나왔으나, 우리가 수집하는 데이터로는 분석이나 ML을 해보기에는 부족하다는 생각이 들었습니다. 
그에 따라 충북대학교 학생들이 자발적으로 데이터를 수집하도록 유도하는 게임을 만들기로 하였습니다.

## 2. 핵심 기능
📊 BLE 센서 데이터 수집
💾 서버 전송 및 저장
🕙 정각 활성화 및 중복 방지
👾 랜덤 캐릭터 출몰
🏆 전송 횟수 기반 랭킹
⭐️ 경험치 및 레벨업

## 3. 시스템 아키텍처
1. Raspberry Pi Sensors  
-> 온도, 습도, eCO2, TVOC, AQI, RSSI
2. Mobile Application  
-> BLE 스캔, 센서 데이터 수신, 위치/근접 판단, 캐릭터 출몰
3. Server  
-> 수신 API, 저장, 순위, 경험치, 캐릭터 등장 로직
4. Database  
-> 센서, 사용자, 캐릭터, 랭킹(단과대/학과/개인)
5. App Screen / Feedback  
-> 센서 정보, 알림, 기록, 랭킹, 레벨
